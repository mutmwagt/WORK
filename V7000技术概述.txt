V7000技术概述(来源: IBM Knowledge Center)
Storwize V7000 Unified
	IBM® Storwize® V7000 Unified 系统是一个虚拟化 RAID 存储系统，同时支持块协议和文件协议。此统一系统包含 IBM Storwize V7000 文件模块(块协议)和 Storwize V7000 存储系统(文件协议)。
	文件模块和存储系统分别包含现实硬件概念和抽象虚拟化概念.
	卷存储系统提供2种模式(虚拟化)的文件系统,一种是 V7000 存储系统的文件模块(虚拟化) 向主机提供网络层级(NFS\CUFS\HTTPS\FTP\CSP\management)的服务(NAS),另一种是主机通过光纤(FCP)向网络提供的硬件层级(虚拟化)的数据库存储服务(DAS).

	-1.硬件
		文件模块的硬件是V7000存储系统提供的,包括2个机柜单元.
		-1.1.控制机柜
			控制机柜包括2个节点容器和1个存储阵列,以及电源模块,备用电池等其他组件.
			-1.1.1.节点容器
				节点容器包含1个系统磁盘和高阶的高速缓存(内存),以及CIMOS电池,芯片等其他组件.
				-1.1.1.1.系统磁盘
					系统磁盘标配是 HDD, 不过由于速度过慢,现在已经升级到 SSD.
				-1.1.1.2.高速缓存
					高速缓存具有易失性,当断电后,缓存数据将慢慢消失,如果电源还在,数据会转移到 SSD 中.
				-1.1.1.3. I/O 组
					2个节点组成 I/O 组为存储系统提供卷存储系统的管理
			-1.1.2.存储阵列
				存储阵列包括12(24)个 SAS 磁盘(SSD,HDD可以混合使用),当存储阵列中含有 SSD 时,会用到分层存储的功能( Easy Tier function).
				-1.1.2.1.RAID 
					系统支持 RAID 0,1,5,6.
				-1.1.2.2.
			-1.1.3.备用电池
				1代备用电池在电源模块上,2代在节点容器内.
		-1.2.扩展机柜
			扩展机柜包括扩展容器和存储阵列,没有节点容器和高速缓存,所以并没有备用电池.
			-1.2.1.扩展容器
				扩展容器包括串行连接 SCSI(SAS)接口硬件,连接到控制机柜上.
		
	#1.V7000的存储区域网络(SAN)之网络接入服务器(network access server NAS)
		储阵列直连网络提供网络层协议的服务所需要的数据存储,服务器主机连接控制机柜提供 I/O和管理.
	#2.V7000的存储区域网络(SAN)之数据库服务(Database as a server DAS)
		和冗余保护,管理等功能,控制机柜直连服务器主机,服务器主机连入网络,提供安全的数据库服务.
	#3.V7000的网络 top 架构
		控制机柜与扩展机柜通过2个6GbpsSAS连接
		控制机柜与2个文件模块分别通过2个8GbpsFC 连接
		控制机柜的 iSCSI 协议映射通过文件模块的4个10GbpsEthernet间接连接
		控制机柜的 FCP   协议映射通过4个8GBpsFC 连接(SAN)
		控制机柜的 iSCSI 协议映射和管理服务通过4个1GbpsEthernet连接
		2个文件模块的 NFS,CIFS,HTTPS,FTP,SCP 协议映射和管理服务分别通过1GbpsEthernet或10GbpsEthernet连接

	-2.软件
		-2.1文件模块软件包
			文件模块软件包是 Storwize V7000 Unified 系统提供的,包括 Interface Node 功能和 Management Node 功能.
